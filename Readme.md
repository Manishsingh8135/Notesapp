
# Notes app

This is simple notes-app with very clean and simple design.
 
## Features 
- Creating new notes
- Editing notes
- Deleting notes


## Run Locally

Clone the project using following command:

```bash
  git clone https://github.com/Manishsingh8135/Notesapp
  
```
